# Servicio Técnico de Computadoras

Sitio web optimizado para la promoción de servicios técnicos y reparación de computadoras.

## Características
* Optimizado minuciosamente para dispositivos móviles.
* Secciones de servicios, tarifas, proceso y contacto directo por WhatsApp.
* Diseño premium oscuro con animaciones y barra de navegación con progreso.

## Tecnologías
* React + Vite
* CSS nativo
